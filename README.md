# CurrencyBridge — Интеграционный сервис курсов валют (pet-project)

## Кратко
CurrencyBridge предоставляет унифицированный REST API для получения курсов валют, агрегирует данные от внешних провайдеров, кеширует ответы и хранит историю. Основными потребителями являются сторонние сервисы, которым необходимы курсы валют в едином формате (например, платежные сервисы или внутренние расчетные модули).
Вторичный пользователь - люди, выполняющие разовые, редкие запросы для получения актуального курса (например, бухгалтер или аналитик).

## Содержимое репо
- openapi.yaml — спецификация API (OpenAPI)
- dbdiagram.dsl — модель данных (dbdiagram.io)
- bpmn/ — BPMN-диаграммы (PNG)
- figma_screenshots/ — прототипы интерфейса
- docs/ — user stories, NFR, error-handling

## Как смотреть
Откройте openapi.yaml в https://editor.swagger.io, dbdiagram.dsl — в https://dbdiagram.io, BPMN — в bpmn/.

## Примечание
Проект учебный. В документации описаны сценарии, модель данных и поведение при ошибках, пригодные для демонстрации на собеседовании.
